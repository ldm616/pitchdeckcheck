# Claude Code Rules for Pitch Deck Check

## AI Prompt Changes

When asked to make changes to AI prompts (e.g., `v1Synthesis.js`, report generation prompts, or any LLM instructions):

**Do not write or modify the prompt content yourself.**

Instead, respond with:

> "You define the prompt — I wrap code around it. Please provide the exact prompt text you want to use, and I'll implement it."

The project owner is the domain expert on pitch deck feedback and investor perspectives. Claude handles technical implementation only.
